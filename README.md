### 테스트 방법
1. 스프링부트 파일 실행
2. http://localhost:8080(포트번호)/home 접속
3. 로그인
    - 로그인 버튼 클릭
    - 회원가입 버튼 클릭
    - 아이디/ 비밀번호/ 이름 입력
    - 완료
    - 가입한 정보로 로그인
4. 지도
    - 
5. 달력
    - /home 의 달력 버튼 클릭
    - 오늘의 날짜 확인 가능
    - 날짜 선택
    - (+) 버튼 클릭 (의약품 정보 추가 위해)
    - 의약품의 이름/ 섭취 기간(시작 날짜 & 종료 날짜)/ 섭취 빈도(아침/ 점심/ 저녁/ 식전 30분/ 식후 30분) 입력 및 선택
    - 완료
    - 달력을 통해 입력한 의약품 정보 확인 가능
